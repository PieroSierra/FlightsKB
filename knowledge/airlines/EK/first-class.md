## First Class Suites

**Claim type:** fact
**Applies to:** airline=EK, cabin=first
**Summary:** Emirates offers private suites with closing doors, onboard shower, and chauffeur service.

Emirates First Class on the A380 features fully enclosed private suites with floor-to-ceiling sliding doors. Each suite includes a personal mini-bar, vanity mirror, and up to 32-inch TV screen.

## Onboard Shower Spa

**Claim type:** fact
**Applies to:** airline=EK, cabin=first, aircraft=A380
**Summary:** A380 First Class passengers can book a 30-minute shower spa session.

The A380 features two onboard shower spas available exclusively to First Class passengers. Each passenger can book one 30-minute session during the flight through the cabin crew.
